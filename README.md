# My Personal Links Page

This repository contains my personal navigation page, consolidating all my social media profiles, portfolios, and active web links in one convenient location.

## ℹ️ About This Repository
* **Original Project:** This site is built using **[LittleLink](https://littlelink.io/)**, an amazing, open-source DIY Linktree alternative created by **[Seth Cottle](https://github.com/sethcottle)**.
* **Modifications:** The core layout, design, and performance optimizations belong entirely to the upstream repository. I have used the codebase exactly as provided, only swapping the default configurations to feature my personal profile images, biographical text, and specific social links.

## 🛠️ Tech Stack & Features
* **Framework-free:** Pure, lightweight HTML and CSS with zero JavaScript bloat.
* **Performance:** High-speed page loading optimized for mobile and desktop screens.
* **Theming:** Built-in support for responsive device themes.
